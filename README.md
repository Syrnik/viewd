# Количество просмотров — плагин для Shop-Script

[English version](README.en.md)

Плагин считает количество просмотров карточки товара и отображает счётчик в бэкенде и на фронтенде магазина.

## Как это работает

Просмотр фиксируется через AJAX-запрос, который отправляется со страницы товара после её загрузки. Это исключает ложные засчитывания при серверном рендеринге. Просмотры не засчитываются, если у посетителя отключены JavaScript или cookies. Авторизованные пользователи бэкенда в счётчик не попадают.

> Отображаемое значение — это число **предыдущих** просмотров; текущий визит будет учтён при следующей загрузке страницы.

## Возможности

- Счётчик просмотров в бэкенде Shop-Script — на странице редактирования товара, вкладка «Цены, наличие, публикация»
- Опциональное отображение счётчика на фронтенде магазина в боковой колонке карточки товара
- Переопределение шаблона вывода из темы оформления

## Настройки

В настройках плагина одна опция — **«Стандартное отображение на карточке товара»**.

Если включена, счётчик выводится через хук `frontend_product.block_aux` (боковая колонка карточки товара). Отключите её, если хотите вывести счётчик в другом месте шаблона вручную.

## Вывод счётчика вручную

Если стандартное отображение отключено, добавьте вызов в шаблон темы оформления.

Для Shop-Script 8.17 и новее:

```smarty
{$wa->shop->viewdPlugin->views($product)}
```

Для старых версий Shop-Script:

```smarty
{shopViewdPluginViewHelper::productViews($product)}
```

## Кастомизация шаблона

Чтобы изменить вёрстку счётчика, добавьте файл `plugin.viewd.product.html` в папку своей темы оформления. Доступные переменные:

| Переменная      | Описание                         |
|-----------------|----------------------------------|
| `$total_views`  | Количество просмотров (целое)    |
| `$product`      | Объект или массив товара         |

## Системные требования

- PHP ≥ 7.4
- Shop-Script (любая актуальная версия)
- Webasyst Installer ≥ 3.0
